=============================AutoGrading Demo with Firebase=============================
This demo analyses 6 exam paper.
This demo will get the hand written student name and the exam score and store the information to firebase.
=======================================================================
This demo should be runned under Python 3.
=======================================================================
To run this demo, you need a firebase project and a json file containing private key to access you firebase project.
=======================================================================
You can get required config imformation to access your firebase following instruction here:
https://www.appypie.com/faqs/how-can-i-get-api-key-auth-domain-database-url-and-storage-bucket-from-my-firebase-account
=======================================================================
Also, you need to install following Python library:
    beautifulsoup4==4.7.1
    Flask==1.0.2
    Flask-Bootstrap==3.3.7.1
    Flask-Dropzone==1.5.3
    Flask-Mail==0.9.1
    Flask-WTF==0.14.2
    lxml==4.3.2
    mysql-connector==2.2.9
    numpy==1.16.2
    opencv-python==4.1.0.25
    pytesseract==0.2.6
    sshtunnel==0.1.4
    pyrebase or pyrebase4
